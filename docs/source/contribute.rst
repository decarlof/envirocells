==========
Contribute
==========

If you are familiar with `sphinx readthedocs <http://docs.readthedocs.io/en/latest/getting_started.html>`_ and `github <https://github.com>`_ you can add a new environment cell or infractructure `Envirocells <https://github.com/decarlof/envirocells>`_  by starting a new `pull request <https://github.com/decarlof/envirocells/pulls>`_. 

If you are not familiar with these tools send an email to envirocells@anl.gov including a description using `this template <http://envirocells.readthedocs.io/en/latest/source/environment>`_ and a 
`pull request <https://github.com/decarlof/envirocells/pulls>`_ will be opened for you.
